# 🎵 音乐播放器使用说明

## 如何添加音乐文件

### 1. 准备音乐文件
将您的音乐文件放入 `web_finalexam/music/` 文件夹中。

**支持的音频格式：**
- MP3 (.mp3) - 推荐
- WAV (.wav)
- OGG (.ogg)
- M4A (.m4a)

### 2. 文件命名建议
建议使用简单的英文文件名，避免使用中文或特殊字符：
```
song1.mp3
song2.mp3
song3.mp3
```

### 3. 修改播放列表

在每个页面的 JavaScript 代码中找到 `playlist` 数组，修改为您的音乐信息：

```javascript
const playlist = [
    {
        title: '你的歌曲名称',
        artist: '艺术家名称',
        src: '../music/song1.mp3',
        cover: '../image/homepic1.png' // 可选：歌曲封面图片
    },
    {
        title: '第二首歌',
        artist: '艺术家2',
        src: '../music/song2.mp3',
        cover: '../image/homepic1.png'
    },
    // 添加更多歌曲...
];
```

### 4. 示例配置


## 功能说明

### 播放器功能
- ▶️ **播放/暂停**：点击中间大按钮
- ⏮️ **上一首**：切换到上一首歌曲
- ⏭️ **下一首**：切换到下一首歌曲
- 🔁 **循环播放**：点击循环按钮，图标变色表示已开启
- 📊 **进度条**：拖动进度条跳转到指定位置
- 📝 **播放列表**：点击列表中的歌曲直接播放

### 打开/关闭播放器
点击导航栏上的音乐图标按钮即可打开或关闭播放器面板。

## 注意事项

1. **文件路径**：确保音乐文件路径正确，相对于 HTML 文件的位置
2. **文件大小**：建议单个音乐文件不超过 10MB，以保证加载速度
3. **版权问题**：请确保您有权使用这些音乐文件
4. **浏览器兼容性**：现代浏览器都支持 HTML5 音频播放
5. **自动播放限制**：部分浏览器可能会阻止自动播放，需要用户手动点击播放

## 快速测试

如果您还没有音乐文件，可以使用免费的测试音频：

1. 访问 https://freesound.org/ 下载免费音效
2. 或使用在线音乐库（注意版权）
3. 将文件重命名为 song1.mp3, song2.mp3 等
4. 放入 `music/` 文件夹
5. 刷新页面即可

## 常见问题

**Q: 音乐无法播放？**
A: 检查文件路径是否正确，浏览器控制台是否有错误提示

**Q: 如何更换封面图片？**
A: 修改 `cover` 字段，指向您的图片路径

**Q: 支持多少首歌曲？**
A: 理论上无限制，但建议不超过 50 首以保证性能

**Q: 如何自动播放？**
A: 浏览器通常会阻止自动播放，需要用户交互后才能播放
